📚 Anime & Mangá App (AngularJS)<br> <br>
📌 Sobre o projeto<br> <br>
Este projeto foi desenvolvido como trabalho final do módulo de AngularJS do curso de ServiceNow da [Faculdade Iv2](https://www.linkedin.com/company/facul-iv2/posts/?feedView=all).<br>
O objetivo principal foi construir uma aplicação web capaz de consumir uma API pública, aplicando na prática os conceitos aprendidos ao longo do curso.<br>
A aplicação é voltada ao universo de animes e mangás, permitindo ao usuário explorar conteúdos populares, buscar obras e salvar seus favoritos.<br>
***
🚀 Funcionalidades <br><br>
🔥 Listagem dos top animes e mangás do momento <br>
🔎 Busca de obras (animes e mangás) <br>
📑 Navegação por acervo <br>
⭐ Sistema de favoritos <br>
💾 Armazenamento de favoritos com Local Storage <br>
📱 Interface simples e intuitiva <br>
***
🌐 API utilizada <br><br>
A aplicação consome dados da API: <br>
Jikan API → https://jikan.moe/ <br>
A Jikan é uma API pública baseada nos dados do MyAnimeList, fornecendo informações atualizadas sobre animes e mangás. <br>
***
🛠️ Tecnologias utilizadas <br> <br>
AngularJS <br>
JavaScript <br>
HTML5 <br>
CSS3 <br>
API REST (Jikan) <br>
Local Storage <br>
***
💡 Decisões de projeto <br> <br>
O uso do Local Storage foi escolhido para armazenar os favoritos do usuário porque: <br>
Não requer backend <br>
É simples de implementar <br>
Atende perfeitamente ao escopo do projeto <br>
